 # Buying Page
 
<div align="center"><a name="menu"></a>

The Sign-up Page consists of 2 main sections:

***1) Header, which includes:***
- Buy button: 
it shows up to you the way to buy a crypto-currency.

- Sell button: 
it redirects you to the selling page.

***2) Buying crypto-currency Form, which includes:***
- How much do you want to buy input field.

- Approximately.

- Receiving address input field.

- Sending account details.

- Select bank input field.

- Account number input field.

- Account name input field.

- Proceed to payment Button: It's a button that redirects you to the payment page.

***2) Order details Form, which includes:***
- Reference number.

- Buying amount.

- Approximately.

- Receiving address.

- Receiving account details.

- Account name.

- Bank name.

- Account number.

- Payment timer: it's a timer that counts down the remaining time for paying.

- I've made my payment Button: It's a button that dones the payment.

***### The page is fully responsive to all mobile devices even the small ones. ###***
